# Model correction
Model correction for linear regression to have more accurate predictive power

## Method
- Polynomial transformation
- Log transformation for independent variables
- Log transformation for dependent variable
- Box-Cox transformation

## Result
- Stablize residuals variance and satisfy linear regression assumptions of the model (zero mean, constant variance, normality)
- Model 4 has the best R-squared value (0.805) with the interpretability and validation of assumptions.
